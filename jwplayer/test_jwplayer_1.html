URI rtmp://media stream server ip/stream?dev_id&slot&stream&mode&user&password
参数名 参数描述 参数说明
dev_id 设备ID 可以通过以下方法获知：设备列表-[单击]设备名称-设备ID
slot 通道号 1-x
stream 主（子）码流 主码流：1
子码流：2
mode 预览/回放 预览：live
回放：playback
beg 回放开始时间 UTC格式
end 回放结束时间 UTC格式
user 用户名
password 密码
例1：预览 rtmp://192.168.18.245/stream?dev_id=00310101031111111000002000000000&slot=1&stream=1&mode=live&user=admin&password=12345
例2：回放 rtmp://192.168.18.245/stream?dev_id=00310101031111111000002000000000&slot=2&stream=2&mode=playback&user=admin&password=12345&beg=2016-03-04T08:00:00.000Z&end=2016-03-04T08:03:00.000Z
